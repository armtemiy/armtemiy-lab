# Админ-модуль

## Назначение

Админ-модуль позволяет загружать и экспортировать JSON дерева решений без Supabase. Данные хранятся локально в `localStorage`.

## Как использовать

1. Открой модуль **Админ: Логика** (доступен только для admin ID).
2. Загрузить JSON:
   - через выбор файла (`.json`), или
   - вставить JSON в текстовое поле.
3. Нажми **Применить JSON** — дерево заменится на загруженное.
4. **Экспорт** сохранит текущую версию дерева на диск.
5. **Сбросить на дефолт** вернет дерево из `src/data/diagnosticTree.ts`.

## Шаблон

Готовый пример находится в `assets/diagnostic-template.json`.

## Где хранится

- Ключ `localStorage`: `armtemiy_lab_tree_override`
- После обновления страницы дерево продолжит использоваться.
